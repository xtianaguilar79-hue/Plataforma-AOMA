# SIGCA v0.8.2 - Corrección móvil

- Se corrige pantalla blanca en dispositivos móviles.
- Se eliminan cachés obsoletas de versiones 0.8.x.
- Se versionan CSS y JavaScript para forzar la descarga de archivos actuales.
- Se incorpora un mensaje visible si el arranque falla.
- La instalación PWA queda temporalmente desactivada hasta estabilizar el comportamiento móvil.

# SIGCA · Changelog

## v0.8.1 · Contenido profesional

### Corregido
- Se integra el logo institucional real en la pantalla de ingreso.
- Se conserva el logo en registro, recuperación, panel y área privada.

- Desarrollo completo en ocho módulos, basado en el manual del Programa de Apoyo a la Formación Sindical.

- Preguntas interactivas con devolución inmediata.
- Reflexiones personales guardadas en el dispositivo.
