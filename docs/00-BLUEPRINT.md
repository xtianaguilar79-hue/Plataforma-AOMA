# SIGCA Blueprint

SIGCA es el Sistema Integral de Gestión, Capacitación y Asistencia de AOMA Seccional San Juan.

## Propósito

Fortalecer el trabajo diario de dirigentes y delegados mediante conocimiento organizado, formación y herramientas de gestión.

## Navegación estable

1. Mi Escritorio
2. Institucional
3. Centro de Conocimiento
4. Capacitación
5. Gestión Sindical
6. Mi Perfil

## Principios

- Mobile first.
- Tema claro y oscuro desde el origen.
- Máximo tres clics para llegar a información relevante.
- No duplicar contenido.
- Mantener versiones estables y migraciones documentadas.
- Toda función debe facilitar el trabajo real de un dirigente o delegado.
