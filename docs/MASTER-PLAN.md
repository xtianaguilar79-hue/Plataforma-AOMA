# Campus AOMA Seccional San Juan - Master Plan

## Propósito
Construir una plataforma institucional de capacitación, consulta normativa y gestión del conocimiento para dirigentes y delegados de AOMA Seccional San Juan.

## Roles
- Administrador: aprueba cuentas y administra usuarios, empresas, convenios y contenidos.
- Dirigente: accede a formación y recursos asignados.
- Delegado: accede a formación y recursos asignados.

## Módulos consolidados
1. Autenticación y recuperación de contraseña.
2. Registro sujeto a aprobación administrativa.
3. Panel de administración de usuarios.
4. Dashboard y perfil del usuario.
5. Biblioteca normativa y convencional.

## Próximos módulos
1. Catálogo de cursos.
2. Cursos, módulos y lecciones.
3. Videos de YouTube privados/no listados.
4. Archivos y material descargable.
5. Evaluaciones y seguimiento del progreso.
6. Certificados verificables con QR.
7. Asistente virtual con fuentes institucionales.
8. Estadísticas y tablero gerencial.

## Reglas de desarrollo
- Cada entrega se distribuye como ZIP completo y versionado.
- Los módulos funcionales no se modifican sin una razón concreta.
- Toda modificación se registra en CHANGELOG.md.
- Las credenciales administrativas nunca se incluyen en el frontend.
- Los permisos dependen de Supabase Auth y Row Level Security.
